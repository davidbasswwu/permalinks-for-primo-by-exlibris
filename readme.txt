Permalinks for Primo
David Bass
WWU Libraries
19 March 2014

Requirements:
- Apache web server
	- ability/permissions to add/use .htaccess (or similar technique in httpd.conf)
- PHP

Current features:
- permalink to a search results page
- permalink to a display page
	
Coming soon:
- add facets to permalink
- add sort-by to permalink
- advanced search permalinks

Setup:
- create a folder on your webserver to put the index.php and .htaccess files in
  - we put them in http://library.wwu.edu/onesearch/ 
- customize the index.php file
